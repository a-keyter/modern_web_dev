# ShadCN - An Awesome UI Library

Fortunately for developers who don't have the time or experience to design the components they will use in their apps, there are a number of UI libraries available.

One of the most popular UI libraries for this is [shadcn](https://ui.shadcn.com/)

<br/>

Shadcn gives us a number of 'Components' that we can incorporate into our apps, ranging from inputs, buttons and forms all the way through to to tables and drop down menus.

## Installing ShadCN in a NextJS App

As per the documentation, we can install the library from the terminal within our project:

```sh copy
npx shadcn-ui@latest init
```

You'll then be asked a number of questions, you can acccept the default for each:

```
Which style would you like to use? › Default
Which color would you like to use as base color? › Slate
Do you want to use CSS variables for colors? › no / yes
```

Now, you'll be able to add UI components to your project from shadcn.

Run the following command in project terminal:

```sh copy
npx shadcn-ui@latest add button input card avatar
```

## Using ShadCN Components

Now that you've installed the UI components, you can incorporate them into your page.

```ts filename="page.tsx" copy

import { Card } from "@/components/ui/card";
import { Input } from "@/components/ui/input";
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar";

export default function Home() {
  return (
    <main className="flex min-h-screen flex-col items-center gap-y-4 p-24 w-full">
      <h1 className="text-6xl font-semibold">Hello World!</h1>
      <p className="text-center text-2xl">This is our first NextJS Project</p>

      <Card className="p-4 w-full max-w-2xl flex flex-col justify-center gap-y-6">
        <div className="flex items-center gap-x-4">
          <Avatar>
            <AvatarImage src="https://github.com/shadcn.png" />
            <AvatarFallback>CN</AvatarFallback>
          </Avatar>
          <div className="flex flex-col">
            <h2>Placeholder Name</h2>
            <small>Placeholder Date</small>
          </div>
        </div>
        <div className="flex flex-col gap-y-2 w-full">
          <h2>Post Title</h2>
          <p>
            Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do
            eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim
            ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut
            aliquip ex ea commodo consequat.
          </p>
        </div>

        <hr />
        <div className="flex w-full justify-center items-center gap-x-8">
          <Input className="w-full max-w-xl" placeholder="Leave a comment..." />
          <Button>Comment</Button>
        </div>
      </Card>
    </main>
  );
}
```
