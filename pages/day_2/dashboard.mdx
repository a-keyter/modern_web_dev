# Building a User Dashboard.

Congratulations, you've implemented Authentication using Supabase.

Now, we need a place for the user to go once they've successfully logged in.

Typically, this will be some kind of Dashboard, where the user can see current content.

Let's build the layout for this User Dashboard.

## Hidden Routes - User Layout

In NextJS, we can create hidden routes for users, to help us manage file structure.

In your app directory, create a new folder called `(user)`

Then, in the (user) directory, create a file called `dashboard/page.tsx` and a file called `create/page.tsx`

In each page type rfce, and you'll generate a component.

You can change the contents of the pages to something like this:

```tsx filename="(user)/dashboard/page.tsx" copy
import React from 'react'

function page() {
  return (
    <div>Welcome to the Dashboard</div>
  )
}

export default page
```

```tsx filename="(user)/create/page.tsx" copy
import React from 'react'

function page() {
  return (
    <div>Welcome to the Create Post Page</div>
  )
}

export default page
```

## User Layout

Next up, we'll want to create a UserNav that a user will use to navigate the actions they can take.

Head into your Components folder and create a new file called `UserNav.tsx`

Then smash your rfce to initialise the component:

```tsx filename="components/UserNav.tsx" copy
import React from 'react'

function UserNav() {
  return (
    <div>UserNav</div>
  )
}

export default UserNav
```

## Create a User Layout

Next, you'll want to create a layout for all pages in the (user) route.

Click on the `(user)` folder and then add new file called  `layout.tsx`

Add the following content to your layout file:

```tsx filename="(user)/layout.tsx" copy
import UserNav from "@/components/UserNav";

export default function RootLayout({
    children,
  }: Readonly<{
    children: React.ReactNode;
  }>) {
    return (
        <main className="flex">
        <UserNav/>
        <div className="w-full max-w-6xl">
          {children}
        </div>
       </main>
    
    );
  }
```

Next, we'll update the content of our UserNav to display links to user pages:

```tsx filename="components/UserNav.tsx" copy
import Link from 'next/link';
import React from 'react'

function UserNav() {
  const UserPages = [
    {
      title: "Dashboard",
      href: "/dashboard"
    },
    {
      title: "Create Post",
      href: "/create"
    }
  ];

  return (
    <div className='w-[250px] p-4 flex flex-col gap-y-4 border-r-2 border-t-2 rounded-tr-lg border-slate-500 h-screen'>
      <p className='font-bold'>User Navigation</p>
      {UserPages.map((page, index) => (
        <Link
          key={index}
          href={page.href}
          className='p-2 rounded hover:bg-slate-400 transition-colors duration-300'
        >
          {page.title}
        </Link>
      ))}
    </div>
  )
}

export default UserNav

```
